# ✨ Design Principles

1. 🟨 Yellow owns every primary action.
2. 📈📉 Green/Red only indicate market movement.
3. 🌑 Dark-first brand experience.
4. ☀️ Light transactional workflows.
5. 📊 Information-dense interfaces.
6. 🎭 Flat surfaces over shadows.
7. 🔢 Numbers feel precise and financial.
8. 🛡️ Trust is always visible.
9. 📱 Mobile-first trading experience.
10. 🚀 Consistent across all crypto workflows.
